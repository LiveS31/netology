                Flask
-микроframework позволяющий писать http сервисы
используется:
    - микросервис
    - Небольшие API
    - MOCK API (симуляция чего - то)
самый распространенный стандарт API - REST API
Запросы
GET - получить
POST - Вставить данные в базу
PUT - Полность изменить запись в базе
DELETE - Удалить запись из базы
Ответы
200 - ок
400 - Неправильная команда
401 - Ошибка авторизации
404 - Нет такой записи в базе
500 - Внутренняя ошибка
Из чего состоит запрос
uri (query string) - возможность передавать ксловари в запросе)
